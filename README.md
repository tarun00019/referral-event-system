# 🚀 Referral & Event Management System

## 📌 Project Description

This is a Full-Stack Referral & Event Management System developed as part of an internship assignment.

The system allows users to:
- Register & Login securely
- Generate and share referral links
- Refer other users
- Create and manage events
- Upload event images
- Track referrals
- View personal dashboard
- Admin monitoring & analytics

---

# 🛠 Tech Stack

### Backend
- Node.js
- Express.js
- MySQL
- Sequelize ORM
- JWT Authentication
- bcrypt (Password hashing)

### Frontend
- React.js (Vite)
- Axios
- React Router

---

# 🗄 Database Structure

Tables used:

# 🔐 Features

## Authentication
- User Registration
- User Login
- JWT-based authentication
- Role-based access (User / Admin)

## Referral System
- Unique referral code generation
- Referral link sharing
- Referral tracking
- Referral leaderboard (Admin)
- Referral statistics

## Event Management
- Create event
- Edit event
- Delete event
- Upload multiple images
- SEO-friendly slug
- Pagination & search

## User Dashboard
- View profile
- View referral code
- Copy referral link
- View referral count
- View referred users
- View personal events

## Admin Dashboard
- View all users
- View all events
- View referral statistics
- View leaderboard
