# Referral & Event Management System

A backend API built using Node.js, Express, MySQL, and Sequelize.

## 🚀 Features

### 🔐 Authentication
- User Registration
- Login with JWT
- Password hashing (bcrypt)
- Protected routes

### 🎁 Referral System
- Unique referral code generation
- Track referrals
- Referral count API
- Referral list API
- Leaderboard API

### 🎉 Event Management
- Create Event (Auth required)
- Unique slug generation
- Upload up to 3 images
- Public event listing
- Pagination
- Search functionality
- Get event by slug
- Delete event (removes images from DB & folder)

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MySQL
- Sequelize ORM
- JWT Authentication
- Multer (File Upload)
- Slugify
